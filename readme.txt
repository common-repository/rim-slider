== Description ==

This is a Simple Image Slider Plugin, Which is a WordPress plugin which I developed from few tutorials, I hope to develop more advanced plugins, I hope you will enjoy it.

How to Use:

1. Create a New Post under Rim Slider

2. Add the image links in the fields then publish it

3. use the short code [rimslider id='1' /]

* Replace the ID to your post id.
